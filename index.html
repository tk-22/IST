HTML and CSS Factsheet

Text Emphasis Features

This is a strong element, which can be used to make text bold and to add custom styling to the text

I have used a text align feature to have all text in the center of the page

The Underline element can be used to create emphasis on certain words

Along with a bold element which can emphasis text even more

The font-weight CSS element can also be used to give customisable boldness to a word or element

The font-family element can be used to customise fonts which are imported from google fonts as evident with the title and text

The em tag can also be used to emphasis certain text by italicizing the text

Background Changes

As in the header, the background colour can also be changed along with a background image

Padding can be added to the sides of the document to center it

In my website I have added a background colour to the entire website along with a different colour to the content and header

Class, ID and Divs

Classes can be used to apply similar settings to multiple elements, for example I have used a "Heading" class to all h1 and h2 elements to apply the same font to all (Koulen)

An id can be used only when styling one single element such as a line of text or a heading

I have used an ID to style the button below

Anchor Tags and Links

Anchor Tags can be used to navigate to other HTML pages or other websites

They can be individualy styled to appear like a

Button

By using the background colour feature along with padding and the border-radius element which creats a curved outline

Unordered and Ordered Lists

Unordered Lists can be used to order items in which order is not important

Lists are formatted using the ul (unordered list) tag and inside this the li (list item)

Ordered lists can also be implemented which provide numbers for each list item
